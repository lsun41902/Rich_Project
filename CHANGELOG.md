# Changelog

업데이트 내용

## [1.3.0] - 2026-04-07
### Added (새로 추가된 기능)
- 미국 주식 시장 검색, 위시리스트 추가

### Fixed (버그 수정)
- 내 주식 목표가 변경후 저장 했을때 재부팅 해야 수정 되는 오류 해결 

### Changed (변경 사항)
- Database 달러 개념 추가
- default_tickers_us 테이블 추가
- tickers 테이블에 나중을 위한 구매, 주식 보유량 관한 정보 컬럼 추가 
- 주식 진단 메세지 현황 출력 되도록 변경

## [1.2.0] - 2026-04-02
### Added (새로 추가된 기능)
- 뉴스 본문 크롤링 기능 추가 (`BeautifulSoup` 활용)
- Neo4j `content` 필드 저장 로직 구현

### Fixed (버그 수정)
- `requests.get` 호출 시 딕셔너리가 전달되어 발생하던 `connection adapters` 에러 수정
- 뉴스 리스트 슬라이싱 시 발생하던 `unhashable type` 오류 해결
- 창이 생성되는 시간이 지연되어 `Thread` 비동기로 처리하고 대기 화면에 로딩화면을 넣어 개선 

### Changed (변경 사항)
- 키워드 분석 쿼리 성능 최적화 (인덱스 적용)

## [1.1.0] - 2026-03-25
### Added
- 주가 데이터와 뉴스 데이터 매칭 알고리즘 초안 반영
- 금 정보를 볼 수 있는 메뉴 추가
